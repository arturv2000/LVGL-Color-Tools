# Changelog

## 0.1.0

- Added `LV_COLOR_MAKE(...)` support with color picker editing.
- Added optional decorator-only support for `lv_palette_main`, `lv_palette_lighten`, and `lv_palette_darken`.
- Added extension settings for `LV_COLOR_MAKE(...)` and palette decorator support.
- Added lightweight unit tests for core color parsing and formatting logic.
- Initial MVP scaffold for LVGL color decorators and color picker support.
